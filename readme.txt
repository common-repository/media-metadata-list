Displays a list of metadata in the Media Library list view.

== Description ==

= Display of video & audio =
* Size
* File type
* Length

= Display of image =
* Size
* Original File
* Original Size
* Thumbnails
* Exif
* Collapsed and displayed.

= Display of files other than above =
* Size
* File type

== Installation ==

1. Upload `media-metadata-list` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
